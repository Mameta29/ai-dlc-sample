# unit-task 非機能要件

## パフォーマンス要件

| ID | 要件 | 目標値 |
|---|---|---|
| NFR-TASK-P01 | タスク作成レスポンス | 200ms以内 |
| NFR-TASK-P02 | タスク更新/削除レスポンス | 200ms以内 |
| NFR-TASK-P03 | タスク一覧取得レスポンス | 500ms以内（100件まで） |
| NFR-TASK-P04 | 先延ばし経過率計算 | クライアント側リアルタイム（1分間隔） |

## スケーラビリティ要件

| ID | 要件 | 詳細 |
|---|---|---|
| NFR-TASK-S01 | ユーザーあたりタスク数 | 初期制限なし。1,000件/ユーザーまでパフォーマンス保証 |
| NFR-TASK-S02 | 期限切れバッチ | 全ユーザーの期限切れタスクを1時間以内に処理 |

## セキュリティ要件

| ID | 要件 | 詳細 |
|---|---|---|
| NFR-TASK-SEC01 | データアクセス制御 | RLS: 自分のタスクのみ閲覧・操作可能 |
| NFR-TASK-SEC02 | 入力バリデーション | Zodスキーマで全入力を検証（XSS防止含む） |

## 信頼性要件

| ID | 要件 | 詳細 |
|---|---|---|
| NFR-TASK-R01 | 完了操作のトランザクション | タスク完了 + スコア確定は一貫性保証 |
| NFR-TASK-R02 | 期限切れバッチの冪等性 | 何度実行しても同じ結果 |

## モニタリング要件

| ID | 要件 | 詳細 |
|---|---|---|
| NFR-TASK-M01 | タスク操作ログ | CRUD操作の構造化ログ |
| NFR-TASK-M02 | 期限切れバッチ統計 | 処理件数、所要時間の記録 |

## テスタビリティ要件

| ID | 要件 | 詳細 |
|---|---|---|
| NFR-TASK-T01 | PBTテスト | ステータス遷移、経過率計算のPBT（fast-check） |
| NFR-TASK-T02 | tRPCルーターテスト | 全エンドポイントの統合テスト |
