# SABOROU ペルソナ定義

## Persona 1: タクヤ（メインペルソナ）

| 項目 | 内容 |
|---|---|
| **名前** | タクヤ |
| **年齢** | 32歳 |
| **職業** | フリーランスWebデザイナー |
| **特徴** | 複数クライアントの案件を同時進行。締切が自己管理のため先延ばしが常態化。「あと2日あるから大丈夫」が口癖。 |
| **動機** | 先延ばしをやめたいわけではないが、自分のパターンを理解したい。ゲーム感覚なら使ってみたい。 |
| **ペイン** | 罪悪感で夜眠れない、締切前日に徹夜が常態化、自己嫌悪のループ |
| **ゲイン** | 先延ばしを正当化できる安心感、自分のパターンの可視化、仲間との共感 |
| **利用デバイス** | スマートフォン（メイン）、PC（作業中のサブ） |
| **利用頻度** | 毎日2〜3回（タスク登録時、先延ばし中、寝る前） |

## AIキャラクター定義

### キャラ1: サボロー（デフォルト）
- **トーン**: フレンドリーな悪友
- **口調**: 「いいよいいよ、明日でいいじゃん」「俺もやってないし」
- **役割**: 先延ばしを肯定、罪悪感を軽減
- **データ収集**: 受容パターン、自己生成型言い訳

### キャラ2: ナマケモノ先輩
- **トーン**: ゆるいメンター
- **口調**: 「まぁ焦らなくても…世界は回るよ」「休むのも才能だから」
- **役割**: 不安の軽減、リラックス誘導
- **データ収集**: ステークホルダー階層、重さの主観次元

### キャラ3: サボリスト
- **トーン**: 挑発的なライバル
- **口調**: 「え、まだやってないの？ まぁ俺の方が先延ばし長いけど」「その程度で罪悪感？甘いね」
- **役割**: 軽い煽りで自己像抽出、着火閾値の測定
- **データ収集**: 自己同一性、着火閾値、言語的トリガー
